# Personal Portfolio

This repository contains my personal portfolio website built with React, Vite, and Tailwind CSS.

## About

This project showcases my skills, projects, and experience as a developer.

## Status

Actively maintained. I update this portfolio as I build new projects and improve existing ones.

## Tech Stack

- React
- Vite
- Tailwind CSS
- TypeScript

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
